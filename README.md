# bulk-mail-from-google-sheet
i write this code for sending bulk mail from google sheet 
**Copy the Script:**
   - Open your Google Sheet.
   - Go to **Extensions > Apps Script**.
   - Paste the contents of `Code.gs` into the script editor.
   - Save the project.

change 8 and 9 line column number for mail and names according to your sheet (ex: A1=1, B=2, C=3, ...)
